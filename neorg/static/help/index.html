Special directives¶

.. table-data-and-image:: path [path ...]¶

Search data under the given list of path and show matched data and corresponding image(s).

base : string (newlines removed)
The base path for searching data. Real path to be used will be {BASE}/{ARG} where {ARG} is each value in the arguments and {BASE} is the value of this option.
data : text [, text, ...]
A comma- or comma-separated list of the “dictionary path”. For each path, period-separated path such as ‘a.b.c’ or ‘alpha.0.1’ can be used.
image : text [, text, ...]
A comma- or comma-separated list of path to the images. The path is the relative path from the parent directory of the data file.
widths : integer [, integer...]
A comma- or space-separated list of relative column widths. Note that the first column is data sub-table and the second and the after are the images specified in :image:.
image-{OPTION} : integer:{VAL} [, integer:{VAL} ...]
integer is the index of the image. {VAL} specifies the value of the {OPTION} of the image directive.

Example:

.. table-data-and-image:: my/experiment/2011-02-*/data.json
   :data: x y result sub.result
   :image: x_y_plot.png x_result_plot.png
